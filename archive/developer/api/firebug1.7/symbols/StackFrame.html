getLineNumber()
Returns the line number within this stack frame's CompilationUnit that is executing. This is where execution has suspended. For the top stack frame, the line number represents the next line to be executed. For frames deeper in the stack it represents the line that is currently executing but has not yet completed.

This function does not require communication with the browser.

Returns:
line number
