Class CrossfireModule

Firebug Module for Crossfire. This module acts as a controller between Firebug and the remote debug connection. It is responsible for opening a connection to the remote debug host and dispatching any command requests to the FirebugCommandAdaptor.

This module also adds context and debugger listeners and sends the appropriate events to the remote host.

Defined in: crossfire.js.
