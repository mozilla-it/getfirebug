Export HTTP Logs

  • The log format is: HAR
  • HAR is supported by many tools:
    • Firebug/NetExport
    • HttpWatch
    • PageSpeed (HAR2PS)
    • ShowSlow
    • DebugBar
    • WebPagetest
    • GTmetrix


HAR allows interoperatibility among HTTP tracing tools.
