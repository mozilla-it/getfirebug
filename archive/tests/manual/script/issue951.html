Issue 951: Code breaks on phantom breakpoint in 1.2.0b6

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Reload the page
  4. Set a breakpoint at line 10 (var output = document.getElementById("output");)
  5. Click the Say delayed hello button and wait for one second
    ⇒ Script execution is stopped at the breakpoint
  6. Click the Continue button inside the Script panel
  7. Remove the breakpoint again
  8. Click the Say delayed hello button again and wait for one second

Expected result

  • Script execution is not stopped anymore
