Issue 3257: Removing all error breakpoints hangs Firefox

  1. Enable all panels using the menu option of the Firebug Status Bar Icon
  2. Switch to the Console Panel
  3. Reload the page
  4. Click on the breakpoint (Break On This Error) besides the error message inside the Console
  5. Switch to the Breakpoints Side Panel inside the Script Panel
  6. Choose Remove All Breakpoints from the Options Mini Menu of the Breakpoints Side Panel
  • Observed: Firefox hangs
  • Expected: The error breakpoint should be removed
