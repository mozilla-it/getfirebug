Issue 4578: Issue 4578: Proxy elements are incorrectly positioned, if page is scrolled

Steps to reproduce

  1. Open Firebug
  2. Start inspecting by clicking the Inspect button
  3. Hover the text input field (#disabledTextField)
  4. Hover the drop-down list (#disabledDropDownList)

Observed result

  • The proxy elements are influenced by scrolling
  • The proxy elements are influenced by <body> positioning styles like left and margin-left etc.

Expected result

  • The proxy elements must exactly overlay the form fields
