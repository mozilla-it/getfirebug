A file with a long name that AJAX loads a file with an error.
  1. Check that the html file name on the Script panel does not spill the toolbar UI (Debugger controls visible)
  2. Check the tooltip on the script panel, should be long
  3. Check the Script Panel popup, the complete filename should show
  4. Check the Console Panel
  5. Set Script->Options->BreakOnError, reload, and check the Stack side panel. Name will be clipped, not wrapped
