Firebug Issue 2645: Edit HTML without having to switch to "Edit" mode

Adding elements to HTML element

  1. Inspect the yellow DIV above (#emptyElement)
  2. Inside the HTML panel try to add elements to the DIV without switching to "Edit" mode
    • Observed: There's no possibility to add elements
    • Expected: The context menu of the DIV should show options for adding child and sibling nodes

Renaming HTML element

  1. Inspect the yellow DIV above (#emptyElement)
  2. Inside the HTML panel try to change the name of the tag without switching to "Edit" mode
    • Observed: There's no possibility to rename the tag
    • Expected: When clicking on the tag's name or pressing F2 while the element is selected a text field should occur containing the current name (like for attributes), that lets you enter a new name

Adding content to empty element

  1. Inspect the yellow DIV above (#emptyElement)
  2. Inside the HTML panel try to add some text to the tag without switching to "Edit" mode
    • Observed: There's no possibility to add text
    • Expected: When clicking between the two arrow brackets ("><") of the empty DIV a text field should occur there, that lets you enter a text (or new tags)
