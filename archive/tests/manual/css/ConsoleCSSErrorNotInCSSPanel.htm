No such line in the CSS Tab

This is a firebug test case for this thread.

What Happened

Notice that in firebug, you will get a message like this:

ConsoleCSSErrorNotInCSSanel.htm (line 20) ! Unknown property 'guinea-pig'. Declaration dropped.

If you click on the link at the left, then it opens up the CSS tab in firebug, however there is no reference to guinea-pig.

What was expected

The link should go to someplace that shows the broken CSS. Either the CSS tab should show broken CSS (perhaps highlighted somehow) or it should show the source of the post-rendered css (with JavaScript DOM changes).

The Firefox error-message line-number is relative to the style element but there is no information in the error as to *which* style element. The test case has its error in the second style, but I can't find that out. Its a bug in FF. A workaround would compare the error message text to each line of source, but I don't know how well that would work vs how hard it would be. A user workaround would always add style elements above any existing style elements.
