Issue 902: CSS Rule Modification

Test Data

This should have a red background initially.

This should have a red background with bold text after testing.

Instructions

  1. Click on the .exists CSS rule
  2. Modify the selector to be .exists, p
  3. Right click at the bottom of the stylesheet view and create a new rule
  4. Set the selector for this rule to be .notExists
  5. Add the following property font-weight: bold
