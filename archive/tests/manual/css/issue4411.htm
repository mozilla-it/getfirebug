  1. Switch to the CSS Panel
  2. Hover the values of the background-color styles of #hex, #name, #rgb, #rgba, #hsl and #hsla
  • Observed: As long as the values are displayed in hexadecimal or rgb format or color names are used infotips are shown, but not for rgba, hsl and hsla color values.
  • Expected: Infotips should always be shown regardless of how the color was specified. Colors with alpha channel should have a background pattern to make the transparency more visible.
