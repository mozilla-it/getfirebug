No such line in the view-source window

This is a firebug test case for this thread.

What Happened

Notice that in firebug, you will get a message like this:

bug1.html (line 3) ! Unknown property 'guinea-pig'. Declaration dropped.

If you click on the link at the left, then it opens up the view source window and shows line three. But since the style is added dynamically via JavaScript, the line is not there.

What was expected

The link should go to someplace that shows the broken CSS, preferably in a way the author could track back to the JavaScript.
