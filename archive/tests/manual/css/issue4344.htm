Issue 4344: CSS autocompletion has incomplete support for color names of CSS3/SVG spec

  1. Inspect the yellow div (#color)
  2. Inside the Style Side Panel edit the background-color value and type deepsky
  • Observed: There is no autocompletion for this color and several other ones of the CSS spec, though there is support for colors, that are not in the spec and not supported by the browser like feldspar.
  • Expected: Autocompletion should complete the entered word to deepskyblue.
