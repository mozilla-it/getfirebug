Eventbug: window listener

This test-case is intended for window listeners
Jan Odvarko, odvarko@gmail.com

  1. Install Firebug
  2. Install Eventbug (Firebug extension)
  3. Open Firebug UI (press F12)
  4. Enable the Script panel and reload the page (in order to see the source code for event listeners).
  5. Select th Events panel.
  6. You should see load event listener
  7. Expand the handler. You should see an event handler with the following implementation: { console.log("Page loaded!"); }
  8. Click the source code snippet, you should be navigated to the Script panel and see the full soure code (starting at line 10).

Some event listeners comes from Firebug itself, they shouldn't be displayed.
