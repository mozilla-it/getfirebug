Issue 2994: "Break On Next" is not working when being clicked while script execution is stopped

  1. Enable and activate the script panel
  2. Reload the page
  3. Click the "Break On Next" button of the script panel
  4. Click the "Change background color" button
  5. Click the "Break On Next" button again
  6. Click the "Continue" button or press F8
  7. Click the "Change background color" button again
  • Observed: The "Break On Next" button is armed, but the execution of the script isn't stopped when clicking on "Change background color"
  • Expected: Clicking on "Change background color" should stop the execution again at line 54 (like it's done for the first time)
