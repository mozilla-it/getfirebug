Issue #2971

This test-case is intended for Issue #2971 - Function declarations not hoisted in console.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Execute following script in the command line:

    var f = function() { return 1; };
    function f() { return 2; }
    alert(f());

    eval(
        'var f = function(){ return 1; };' +
        'function f(){ return 2; }' +
        'alert(f());'
    );
  3. There should be two alerts showing 1 and 1
