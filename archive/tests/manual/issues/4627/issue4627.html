Issue 4627: String.prototype methods not showing up

Steps to reproduce

  1. Open Firebug
  2. Switch to the DOM panel
  3. Inside the DOM panel's options menu uncheck Show Own Properties Only and Show Enumerable Properties Only
    ⇒ The DOM panel should list all properties (enumerable and non-enumerable own properties and prototype chain properties) of window
  4. Check the Show Own Properties Only option
    ⇒ The DOM panel should just list all properties (enumerable and non-enumerable), that directly belong to the window object
  5. Check the Show Enumerable Properties Only option
    ⇒ The DOM panel should just list enumerable properties, that directly belong to the window object
  6. Uncheck the Show Own Properties Only option
    ⇒ The DOM panel should just list all enumerable properties (own properties and prototype chain properties)

Observed result

  • If Show Own Properties Only is checked, the prototype chain is displayed
  • If both options (Show Own Properties Only and Show Enumerable Properties Only) are unchecked, only enumerable properties are shown
  • The option Show Enumerable Properties Only can't be check independently of Show Own Properties Only

Expected result

  • Both options should be available independently from each other
  • The DOM panel display needs to be adjusted according to the selected options
