Issue #3632

This test-case is intended for Issue #3632 - Script error when extending String.prototype and enabling the Console panel.
Jan Odvarko, odvarko@gmail.com

  1. Install extension attached to bug 610973
  2. Open Firefox -> Tools -> Error Console
  3. Click on the Execute Test button below.
  4. You should see following error: "hello".foo is not a function


The Javascript that caused the error looks like as follows: 
function onExecuteTest()
{
    String.prototype.foo = function() {}
    "hello".foo();
}
