Issue 4009: XHR messages are different in the Net and Console panel

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Reload the page
    ⇒ Two XMLHttpRequest logs to issue4009.php are shown.
    ⇒ The time shown for the second request does not correspond to the times shown inside the Net panel.
  4. Expand both log entries
    ⇒ The responses get mixed up between the both requests.

Observed result

  • The request logs are incorrectly displayed.

Expected result

  • The requests should be shown like inside the Net panel, i.e.
  • the response for the second request should be Hello Firebug user!
  • the request time should be the same as inside the Net panel
