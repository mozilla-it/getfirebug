Issue 6434: Very long URLs cause horizontal scroll bar to appear

Steps to reproduce

  1. Open Firebug
  2. Switch to the Net panel
  3. Reload the page
    ⇒ The last request should be for request?extremelyLongParam...
  4. Hover the request's URL

Observed result

  • A horizontal scrollbar appears because the panel tries to display the whole URL string.

Expected result

  • No scrollbar should appear, instead the URL should be cropped or the rest of the URL just hidden.
