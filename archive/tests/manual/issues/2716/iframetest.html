Issue #2716

This test-case is intended for Issue #2716 - Script source does not refresh when reloading an iframe.

  1. Disable cache.
  2. Click 'open frame' button.
  3. select test1.js in script pane.
  4. set a breakpoint at 'var c = 1' line.
  5. Click test button
  6. Verify it stopped at the break point, then Continue.
  7. click 'close frame'
  8. add a line after 'var c = 1;'. make it 'alert('hi');' Save file.
  9. click open frame.
  10. At this point, test1.js is out of date. If you switch to test1.html, then back to
  11. test1.js, it will update.
