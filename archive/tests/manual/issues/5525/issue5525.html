Issue 5525: Setting error breakpoints causes error in Breakpoints side panel

Steps to reproduce

  1. Open Firebug
  2. Enable the Script and the Console panel and switch to the Console panel
  3. Reload the page
    ⇒ The console must list a syntax error: undefinedVariable is not defined
  4. Click the breakpoint besides the code sample (var test = undefinedVariable;)
  5. Switch to the Script panel and there to the Breakpoints side panel

Observed Result

  • The Breakpoints side panel is empty
  • The Firefox Error Console lists:
    Error: TypeError: analyzer.getBaseLineNumberByScript is not a function Source File: chrome://firebug/content/js/sourceFile.js Line: 928

Expected Result

  • The Breakpoints side panel should list the error breakpoint and not cause any error by itself.
