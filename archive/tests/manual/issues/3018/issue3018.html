Scroll Speed

Test Case for Script panel scrolling speed.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Sript panel.
  2. Reload this page
  3. Select script1.js from the file menu (100000 lines).
  4. Try to scroll vertically, the scrolling should be fast/smooth enough.
  5. Select script2.js and check the speed, should be again fast (1440 lines only, but some lines are long).
  6. Now open frame3018.html (30008 lines) scroll somewhere to the bottom (try more times) and compare the speed. Scrolling in this file is reasonably slower (freezing sometimes) since there is a lot of inner scripts.
