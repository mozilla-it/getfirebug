This test-case is intended for Issue #4985 - XHR Spy: An attempt was made to use an object that is not, or is no longer, usable.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and select/enable the Console panel.
  2. Make sure you have Show XMLHttpRequests option checked.
  3. Press the Execute Request button.
  4. An error is displayed: An attempt was made to use an object that is not, or is no longer, usable
