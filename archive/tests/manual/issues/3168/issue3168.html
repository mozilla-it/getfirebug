Issue 3168: Guess the line number of persistent breakpoints

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Reload the page
  4. Set a breakpoint at line 11 (output.textContent = "Hello Firebug user!";)
  5. Open the file in an editor, add the following code before line 11 and save the file:
    var placeholder1 = 1; var placeholder2 = 2;
  6. Reload the page again

Expected result

  • The breakpoint should now be moved to line 13 and not line 11 anymore.
