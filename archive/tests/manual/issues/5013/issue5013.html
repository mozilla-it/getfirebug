Issue 5013: Edit button doesn't work when there's no CSS at all

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Click the Source Edit button
    Nothing happens
  4. Change to Live Edit
  5. Click the Live Edit button
    Nothing happens

Expected Results

  • The Source Edit option should be hidden, because there is no source to edit
  • The Live Edit option should inject a new stylesheet like it's done when clicking on "create a rule" with the Edit Mode off
