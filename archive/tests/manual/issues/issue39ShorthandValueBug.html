CSS Shorthand Style getPropertyValue() Bug

CSS property border-top is a short hand for three properties, border-top-style border-top-width, and border-top-color.

When CSS property border-top-style is set, then the CSS short hand property border-top should have its style value equal to the value of border-top-style. This page has an onload method to test the properties and put the results on this page and output via window.dump to the console.

This issue came up in Firebug as Issue 39.
