Issue 4738: "Open in new tab" opens wrong url when reusing XMLHttpRequest objects

Steps to reproduce

  1. Selecte and enable the Console panel
  2. Reload this page
  3. Right click on the first request http://legoas/path1 and pick "Open in New Tab".

Expected result

  • A new tab with the following URL should be opened: http://legoas/path1

Observed result

  • Tab with http://legoas/path2 is opened.
