Bug 609663

This test-case is intended for Bug 609663 - JSD doesn't support jsdIDebuggerService.interruptHook.
Jan Odvarko, odvarko@gmail.com

  1. Install extension from Bug 609663
  2. Open Firefox -> Tools -> Error Console. Make sure you have javascript.options.showInConsole set to true.
  3. Click on the Execute Test button below.
  4. You should see following interrupts (lines with the executed code): 
    Error: onInterrupt; Line: 1
Error: onInterrupt; Line: 8
Error: onInterrupt; Line: 10
Error: onInterrupt; Line: 11
Error: onInterrupt; Line: 13
Error: onInterrupt; Line: 1


This does work on Firefox 3.6, but does not on Firefox 4.0
