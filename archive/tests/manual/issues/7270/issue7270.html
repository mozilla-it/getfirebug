Issue 7270: Logging groups containing tables is broken when logging multiple groups in a row

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Reload the page

Observed result

  • There are 17 groups listed inside the Console panel.
  • The last group has a message counter of 4 but expanding it doesn't display any contents.
  • The second group entry contains five tables.

Expected result

  • There should be 20 groups logged and each one should contain just one table.
