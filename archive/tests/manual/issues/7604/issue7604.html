Issue 7604: Unrecognized CSS rule types break the display within the CSS and the DOM panel

CSS panel

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel

Observed result

  • The display of the CSS panel is broken. (No rules are shown and the side panels of the previous panel are displayed.)

Expected result

  • The unrecognized rule should just be skipped when displaying the rules.

DOM panel

Steps to reproduce

  1. Open Firebug
  2. Switch to the DOM panel
  3. Expand the tree document > styleSheets > 0 > cssRules

Observed result

  • The display of the rules is broken. (There's just one empty item instead of two normally displayed ones.)

Expected result

  • The unrecognized rule should just be displayed as CSSRule.
