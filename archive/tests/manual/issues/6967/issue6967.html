Issue 6967: Breakpoints display persists after switching the source file

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Switch to issue6967.js via the Location Menu
  4. Set a breakpoint at line 1 (var hello = "Hello Firebug user!";)
  5. Switch to issue6967.html via the Location Menu

Observed result

  • The breakpoint at line one is still visible.

Expected result

  • The breakpoint should only be displayed for issue6967.js.
