Issue #3475

This test-case is intended for Issue #3475 - Logging calls to function can not be removed.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Put onExecuteTest in the command line and press enter key.
  3. Right click on the result in the Console and pick Log calls to: "onExecuteTest".
  4. Click on the Execute Test button below.
  5. You should see a log informing about the function call.
  6. Switch to the Script panel, select the Breakpoints side panel and remove the breakpoint.
  7. Switch back to the Console panel, press the Test button and make sure the function call logs are not there anymore.
