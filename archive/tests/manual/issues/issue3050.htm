Issue 3050: Add "Persist" button to HTML, CSS and DOM panel for keeping changes after a page reload

Persistent HTML changes

  1. Inspect the button
  2. Inside the HTML Panel change the text content of the button to xyz
  3. Reload the page
  4. Have a look at the button
  • Observed: The changes are gone and there is no option for letting the changes persist
  • Expected: There should be a persist option like in the Net Panel, which keeps the changes after reload

Persistent CSS changes

  1. Inspect the DIV element containing the button
  2. Switch to the CSS Panel/Style Side Panel
  3. Change the background color of the style #content div to #ff0000
  4. Reload the page
  5. Have a look at the background color of the style #content div
  • Observed: The changes are gone and there is no option for letting the changes persist
  • Expected: There should be a persist option like in the Net Panel, which keeps the changes after reload

Persistent DOM changes

  1. Click the button Say something (Hello everybody! will be displayed)
  2. Switch to the DOM Panel
  3. Change the variable text to Hello Firebug team!
  4. Click the button Say something again (Hello Firebug team! will be displayed)
  5. Reload the page
  6. Have a look at the text variable resp. click the Say something button again
  • Observed: The changes are gone and there is no option for letting the changes persist
  • Expected: There should be a persist option like in the Net Panel, which keeps the changes after reload
