Issue 5019: Editing a variable in the Watch window doesn't work

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel and there to the Watch side panel
  3. Reload the page
  4. Create a new watch expression b
    ⇒ The variable value is 20
  5. Double-click its value to edit it
    ⇒ Variable name is displayed in the inline edit instead of the value
  6. Refresh the page
    ⇒ The variable value is ReferenceError: b is not defined instead of 20
  7. Click the Test button
    ⇒ The debugger should break at line 15
  8. Double-click on the value of the variable a
    ⇒ The inline edit displays a10 instead of 10
  9. Change it to 15
    ⇒ The new value should be displayed, but the old one is still there
